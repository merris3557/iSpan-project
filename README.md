【饗島】一站式美食平台
第三方服務平台
  對消費者： 快速尋味、輕鬆預約、質感購物，享受從餐廳到餐桌的完整體驗。
  對商家： 整合資訊發布、訂位管理與電商功能，一站式數位轉型，省去跨平台操作的繁瑣負擔。
三大核心
  導覽（地圖）： 精準標記寶島座標，讓在地美味觸手可及。
  引導（訂位）： 優雅預約舌尖盛宴，縮短您與美味的距離。
  導購（商城）： 一鍵下單島嶼風味，將永續生活直送到家。

● 首頁、「島嶼初心」
前端畫面設計，以簡潔、溫暖畫面傳達品牌核心理念。

● 美味座標 -互動式搜尋地圖
圖層整合與互動
 運用 Leaflet.js 建立輕量化地圖互動介面，整合 OpenStreetMap 圖資，實現多點位標記 (Markers) 與客製化彈窗 (Popups)，提升使用者查找店點的直覺性。
地理轉碼整合
串接 Mapbox Geocoding API，將商家建立的地址精確轉換為經緯度座標並存入資料庫，當一般使用者搜尋到該店址時可從資料庫撈取經緯度，以實作地圖中心自動位移與縮放功能。

● 聯絡我們 -客服中心前台與後台 
雙軌並行的回饋表單
  目的：避免登入流程中斷使用者填寫意願
  提供訪客填寫模式與會員填寫模式。
客服後台管理系統
  提供管理者"狀態篩選"、"回覆"、"刪除"...等CRUD功能。

自動化郵件通知 (SMTP)
  目的：透過即時觸發機制，消弭使用者等待時的資訊不對稱。
  整合 JavaMailSender 與 SMTP 協定，並導入 Google App Password。
  視覺化溝通優化：運用 MimeMessageHelper 封裝 HTML 模板與多媒體資源，將傳統純文字信件轉化為結構清晰、美觀的視覺化介面。
