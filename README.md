# 【饗島】一站式美食平台 ｜ 資展國際 結訓專案
> **整合導覽、訂位與電商的第三方服務平台，協助商家達成一站式數位轉型。**

---

## 饗島三大核心
* **導覽（地圖）：** 精準標記寶島座標，讓在地美味觸手可及。
* **引導（訂位）：** 優雅預約舌尖盛宴，縮短您與美味的距離。
* **導購（商城）：** 一鍵下單島嶼風味，將永續生活直送到家。

---

## 技術主責與模組開發

### 1. 頁面開發與架構
| 功能模組 | 對應組件 / 檔案 |
| :--- | :--- |
| **首頁** | `HomeView.vue` |
| **島嶼初心** | `AboutView.vue` |
| **美食地圖** | `MapSearchView.vue` |
| **意見回饋（前台）** | `FeedbackView.vue` |
| **意見回饋（後台）** | `FeedbackAPView.vue` |

### 2. 互動式搜尋地圖 (美味座標)
* **圖層整合與互動：** 運用 `Leaflet.js` 建立輕量化地圖介面，整合 `OpenStreetMap` 圖資，實現多點位標記 (Markers) 與客製化彈窗 (Popups)。
* **地理轉碼整合：** 串接 `Mapbox Geocoding API` 將商家地址精確轉換為經緯度並存入資料庫，實現地圖中心自動位移與縮放功能。

### 3. 客服中心系統 (前、後台)
* **雙軌並行表單：** 提供「訪客」與「會員」雙模式，避免登入流程中斷使用者填寫意願。
* **管理系統：** 提供管理者狀態篩選、回覆、刪除等完整 **CRUD** 功能。
* **自動化郵件通知 (SMTP)：**
    * 整合 `JavaMailSender` 與 `SMTP` 協定，並導入 Google App Password。
    * **視覺化優化：** 運用 `MimeMessageHelper` 封裝 HTML 模板，將純文字信件轉化為美觀的視覺介面，消弭資訊不對稱。

### 4.  首頁/島嶼初心
*  負責前端畫面設計，以簡潔、溫暖的調性呈現，旨在傳達品牌的核心溫情與在地連結。
